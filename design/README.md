# @design ディレクトリについて

この場所は『The Non-Designer's Design Book』の学習メモと発表準備を集約するためのワークスペースです。次の考え方でファイルを配置しています。

- `structure/`: 発表スライドや進行のアウトラインなど、内容の構成を言語化するためのメモを置く場所。
- `examples/`: 書籍の原則を守っている/守れていないサンプルを同じ題材でHTML化したものを格納。まずは言語化せず見てもらう想定。
  - `examples/good/`: 原則を守ったパターン。
  - `examples/bad/`: 原則が崩れているパターン。

今後は各ディレクトリにファイルを追加しながら、観察→言語化→説明の順に情報を整理していきます。
