# AWS セキュリティの概要

## はじめに

AWSのセキュリティは、AWS Well-Architected Frameworkの5つの柱の一つです。セキュリティのベストプラクティスを実装することで、データ、システム、アプリケーションを保護できます。

## セキュリティの主要なトピック

### 1. アイデンティティとアクセス管理（IAM）
- IAMユーザー、グループ、ロール
- ポリシーと権限管理
- MFA（多要素認証）
- 一時的な認証情報

### 2. ネットワークセキュリティ
- VPC（Virtual Private Cloud）
- セキュリティグループ
- ネットワークACL
- VPNとDirect Connect

### 3. データ保護
- 暗号化（保存時、転送時）
- KMS（Key Management Service）
- Secrets Manager
- データ分類とラベリング

### 4. 監視とロギング
- CloudTrail
- VPC Flow Logs
- GuardDuty
- Security Hub

### 5. コンプライアンス
- AWS Artifact
- コンプライアンスプログラム
- セキュリティベストプラクティス
