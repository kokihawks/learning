# AWS データベースの概要

## はじめに

AWSのデータベースサービスは、様々なデータベースエンジンとユースケースに対応したマネージドサービスを提供します。アプリケーションの要件に応じて、適切なデータベースサービスを選択することが重要です。

## データベースの主要なトピック

### 1. RDS（Relational Database Service）
- RDSデータベースエンジン（MySQL、PostgreSQL、MariaDB、Oracle、SQL Server）
- RDSインスタンスクラス
- マルチAZデプロイメント
- 読み取りレプリカ
- RDSスナップショット
- RDS自動バックアップ

### 2. DynamoDB（NoSQL）
- DynamoDBテーブルとアイテム
- パーティションキーとソートキー
- プロビジョニングされたキャパシティとオンデマンド
- DynamoDB Streams
- DynamoDB Global Tables
- DynamoDB Accelerator（DAX）

### 3. ElastiCache（キャッシング）
- ElastiCache for Redis
- ElastiCache for Memcached
- クラスターモード
- レプリケーショングループ

### 4. その他のデータベースサービス
- Redshift（データウェアハウス）
- DocumentDB（MongoDB互換）
- Neptune（グラフデータベース）
- Timestream（時系列データベース）
- QLDB（台帳データベース）

### 5. データベースの最適化
- パフォーマンスチューニング
- バックアップとリストア
- 高可用性の設計
- セキュリティと暗号化
