# AWS モニタリングとロギングの概要

## はじめに

AWSのモニタリングとロギングサービスは、アプリケーションとインフラストラクチャの可観測性を提供します。適切なモニタリングとロギングにより、パフォーマンスの問題を特定し、セキュリティイベントを追跡し、運用を効率化できます。

## モニタリングとロギングの主要なトピック

### 1. CloudWatch
- CloudWatchメトリクス
- CloudWatchアラーム
- CloudWatchダッシュボード
- CloudWatch Logs
- CloudWatch Logs Insights
- CloudWatch Contributor Insights

### 2. CloudTrail
- CloudTrailトレイル
- 管理イベントとデータイベント
- CloudTrailログの保存と分析
- CloudTrail Insights
- イベント履歴

### 3. X-Ray
- X-Rayトレーシング
- サービスマップ
- トレース分析
- サンプリングルール
- セグメントとサブセグメント

### 4. その他のモニタリングサービス
- VPC Flow Logs
- Route 53 Resolver Query Logs
- Application Load Balancer Access Logs
- S3 Server Access Logs

### 5. ログの集約と分析
- CloudWatch Logs Insights
- Elasticsearch Service（OpenSearch）
- Kinesis Data Firehose
- Athena

### 6. アラートと通知
- CloudWatchアラーム
- SNS通知
- EventBridge
- Lambda関数による自動対応
