# AWS ネットワーキングの概要

## はじめに

AWSのネットワーキングは、クラウドインフラストラクチャの基盤となる重要な要素です。適切なネットワーク設計により、セキュリティ、パフォーマンス、可用性を向上させることができます。

## ネットワーキングの主要なトピック

### 1. VPC（Virtual Private Cloud）
- VPCの設計と設定
- サブネットとルートテーブル
- インターネットゲートウェイ（IGW）
- NATゲートウェイとNATインスタンス
- VPCピアリング
- Transit Gateway

### 2. セキュリティ
- セキュリティグループ
- ネットワークACL
- VPC Flow Logs
- AWS WAF
- Shield

### 3. 接続サービス
- VPN（Site-to-Site VPN、Client VPN）
- Direct Connect
- Transit Gateway
- VPCエンドポイント

### 4. コンテンツ配信
- CloudFront（CDN）
- Route 53（DNS）
- Global Accelerator

### 5. ロードバランシング
- Application Load Balancer（ALB）
- Network Load Balancer（NLB）
- Classic Load Balancer（CLB）
- Gateway Load Balancer
