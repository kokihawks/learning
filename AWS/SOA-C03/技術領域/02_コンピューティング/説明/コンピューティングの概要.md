# AWS コンピューティングの概要

## はじめに

AWSのコンピューティングサービスは、アプリケーションを実行するための多様なオプションを提供します。ワークロードの特性に応じて、適切なコンピューティングサービスを選択することが重要です。

## コンピューティングの主要なトピック

### 1. EC2（Elastic Compute Cloud）
- EC2インスタンスタイプ
- AMI（Amazon Machine Image）
- EBSボリューム
- セキュリティグループ
- Auto Scaling
- Elastic IP

### 2. Lambda（サーバーレス）
- Lambda関数の作成と実行
- イベントソース
- バージョンとエイリアス
- 同時実行数の制限
- VPC統合

### 3. コンテナサービス
- ECS（Elastic Container Service）
- EKS（Elastic Kubernetes Service）
- Fargate
- ECR（Elastic Container Registry）

### 4. その他のコンピューティングサービス
- Elastic Beanstalk
- Lightsail
- Batch
- App Runner

### 5. オーケストレーション
- Auto Scaling
- Elastic Load Balancing
- Spot Instances
- Reserved Instances
