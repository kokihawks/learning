# AWS ストレージの概要

## はじめに

AWSのストレージサービスは、様々な用途に応じたストレージオプションを提供します。データの特性、アクセスパターン、コスト要件に応じて、適切なストレージサービスを選択することが重要です。

## ストレージの主要なトピック

### 1. S3（Simple Storage Service）
- S3バケットとオブジェクト
- S3ストレージクラス
- S3ライフサイクルポリシー
- S3バージョニング
- S3レプリケーション
- S3アクセス制御

### 2. EBS（Elastic Block Store）
- EBSボリュームタイプ
- EBSスナップショット
- EBS暗号化
- EBSパフォーマンス最適化
- EBSマルチアタッチ

### 3. EFS（Elastic File System）
- EFSファイルシステム
- EFSストレージクラス
- EFSパフォーマンスモード
- EFSスループットモード
- EFSアクセスポイント

### 4. その他のストレージサービス
- FSx（Lustre、Windows File Server）
- Storage Gateway
- Glacier
- Backup

### 5. ストレージの最適化
- ストレージクラスの選択
- ライフサイクルポリシー
- データの圧縮と重複排除
- コスト最適化
